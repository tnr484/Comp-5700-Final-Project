# COMP-5700 Final Project
Student: Thomas Richards

Class: COMP 5700 – Secure Software Process

Project: Final Project

This repository contains the code and CSV outputs for the Secure Software Process project. The project mined a dataset of pull requests automatically generated by agentic AI tools and analyzed the security-related tasks performed.
main.py: Python script performing Tasks 1–5 (extracting data, generating CSVs, and flagging security-related PRs).
CSV outputs (task1_…csv through task5_summary.csv) contain the processed data.
Large CSV files are tracked using Git LFS and may not display in the GitHub web interface.
The repository follows the course rubric and contains all code and outputs necessary for evaluation.
